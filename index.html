About

cMath is a web app for performing various math calculations used in construction.

cMath is not a construction tutorial. It is meant for users who are versed in construction.

cMath was created by tone925 and is hosted on GitHub. The URL is:

https://tone925.github.io/cMath/

cMath was created for my own use, however, anyone may use it with the understanding that it is a work in progress and may contain errors.

There is no guarantee of correctness and no warranty of any kind. Use with caution.
